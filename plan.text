  2.增加autolog功能（单次试验的自动日志跟踪）

预处理/过程超参数：
    部分固定为经验值+核心超参网格搜索+次要超参随机搜索
    模型结构超参数：
    在固定其他参数前提下遍历/网格

python_client/
├── core/                  # 核心调优与执行模块
│   ├── experiment.py      # Experiment类（父进程，实验管理）
│   ├── test.py            # Test类（子进程，单次试验）
│   ├── scheduler.py       # 任务调度器（多进程/线程池）
│   └── param_generator.py # 超参数空间生成器（网格+随机）
├── api/                   # 与Java后端交互模块
│   ├── client.py          # HTTP客户端（封装GET/POST/PUT）
│   └── payload_builder.py # 请求体构造（实验信息/测试数据）
├── log/                   # 日志与持久化模块
│   ├── logger.py          # 线程/进程安全日志器
│   └── result_manager.py  # 结果缓存与持久化
├── utils/                 # 工具模块
│   ├── config.py          # 全局配置（并发数、超时、后端地址等）
│   ├── exceptions.py      # 自定义异常类
│   └── helpers.py         # 通用辅助函数
└── main.py                # 入口脚本（启动实验、解析命令）
